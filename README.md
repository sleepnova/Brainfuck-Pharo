# Brainfuck-Pharo
A Brainfuck implementation in Pharo Smalltalk

## Usage:
```Smalltalk
BFRunner run: 'bf code'
```
## Example:
Print "Hello World!" to Transcript.
```Smalltalk
BFRunner run: '++++++++[>++++[>++>+++>+++>+<<<<-]>+>->+>>+[<]<-]>>.>>---.+++++++..+++.>.<<-.>.+++.------.--------.>+.>++.'
```
